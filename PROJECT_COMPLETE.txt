🚀 QUICK START (5 MINUTES)
════════════════════════════════════════════════════════════════════════════════

1️⃣  ACTIVATE VIRTUAL ENVIRONMENT
    $ source .venv/bin/activate

2️⃣  UPDATE .env FILE
    Edit .env with your Azure credentials:
    - AZURE_OPENAI_ENDPOINT
    - AZURE_OPENAI_API_KEY
    - SPEECH_KEY
    - SPEECH_REGION

3️⃣  START LOCAL SERVER
    $ func start
    API runs at: http://localhost:7071/api

4️⃣  OPEN WEB UI
    $ open index.html
    or use: python -m http.server 8000

5️⃣  TEST IT
    • Upload an image
    • Select a concept
    • Generate audio in your language

🔧 API ENDPOINTS
════════════════════════════════════════════════════════════════════════════════

POST /api/upload_notes
  • Upload image of notes
  • Returns: highlighted image + extracted concepts

POST /api/generate_speech
  • Generate audio explanation for concept
  • Returns: audio in base64 format (WAV)

GET /api/supported_languages
  • List available languages for speech synthesis
  • Returns: language codes and voice options

GET /api/health
  • Health check endpoint
  • Returns: service status
